Mystiko Scanner Related APIs.

## Sync

Sync current wallet deposit and spend transaction status.
The ScannerSyncOptions concurrency level is enforced to 1 in the SDK.

```node
import mystiko from '@mystikonetwork/node';
import { core } from '@mystikonetwork/protos';

const options = new core.scanner.v1.ScannerSyncOptions({
    walletPassword: WalletPassword,
});
const balanceResult = mystiko.scanner?.sync(options);
```

The parameter list for `ScannerSyncOptions`:
<Table
    headers={["Parameter", "Required", "Description"]}
    data={[
        {
            'Parameter': { code: 'walletPassword' },
            'Required': 'true',
            'Description': 'The password of the wallet.'
        },
        {
            'Parameter': { code: 'concurrency' },
            'Required': 'false',
            'Description': 'Concurrency level is enforced to 1 in the SDK.'
        }
    ]}
/>

## Scan

Scan transactions related to the current wallet.
The ScannerScanOptions concurrency level is enforced to 1 in the SDK.

```node
import mystiko from '@mystikonetwork/node';
import { core } from '@mystikonetwork/protos';

const options = new core.scanner.v1.ScannerScanOptions({
    walletPassword: 'walletPassword',
    shieldedAddresses: [],
});
const scanResult = mystiko.scanner?.scan(options);
```
